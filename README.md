# Color Palette Generator

## 🎨 Overview

Color Palette Generator is a web-based application that helps designers and developers create beautiful and harmonious color palettes instantly. Built with HTML, CSS, and JavaScript, the application generates random color combinations and allows users to copy color codes for use in websites, mobile apps, branding projects, and graphic design work.

The project showcases practical JavaScript concepts such as dynamic content generation, event handling, clipboard interactions, and responsive user interface development.

## ✨ Features

* Generate random color palettes instantly
* Display HEX color codes
* One-click color code copying
* Responsive and modern user interface
* Clean and intuitive design
* Fast and lightweight performance
* Dynamic color generation
* Mobile-friendly experience
* Easy customization and extension

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Clipboard API

## 🚀 Getting Started

### Prerequisites

To run this project, you only need a modern web browser:

* Google Chrome
* Mozilla Firefox
* Microsoft Edge
* Safari

### Installation

Clone the repository:

```bash
git clone https://github.com/frankpatel1/Color-Palette-Generator.git
```

Navigate to the project folder:

```bash
cd Color-Palette-Generator
```

Open the application:

```bash
index.html
```

Or simply double-click the `index.html` file to launch it in your browser.

## 📖 Usage

1. Open the Color Palette Generator in your browser.
2. Generate a new palette using the provided button or keyboard shortcut.
3. Browse the displayed color combinations.
4. Click a color code to copy it to your clipboard.
5. Use the copied colors in your design or development projects.
